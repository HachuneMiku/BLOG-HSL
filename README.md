# BLOG-HSL



### 项目介绍


#### 服务端依赖的npm包
项目框架：koa
路由系统：koa-router
处理文字请求体数据：koa-bodyparser
生成token的包：jsonwebtoken
处理登录后的session：koa-session
页面渲染：koa-art-template(依赖art-template)
静态资源处理：koa-static
散列密码的库：bcrypt
连接数据库：mysql

时间处理：moment
生成验证码：captchapng2
处理文件上传的：请求体数据  koa-formidable
