项目经验

2018.10 - 2019.4

雄通维修平台以及小程序

该项目是一个在线退运检测维修平台，提供从港口到维修车间再到港口全程服务。前台用户端主要是下单，查询订单，订单取消，支付订单。后台管理端主要是审核订单，生成检查费用，修改订单的状态，上传维修检查报告。有PC端和小程序端。

该项目前后端分离，后台使用TP5来支撑，数据库使用MySQL。而我前端PC部分主要使用vue.js，jQuery来支撑，页面则采用layUI作为支撑，数据交互采用axios来支撑。小程序使用到了colorUI和iview的UI组件库来作为页面的支撑。

我负责前台用户，后台管理员和小程序前端的所有工作，包括页面的搭建，和后台的数据交互等。异步请求全部使用async && await函数在axios的api上重新封装了一遍，文件上传采用了阿里云对象存储OSS，支付功能调用了支付宝接口，电子合同引入了第三方电子签名服务，在线聊天则使用了融云的即时通讯服务。同时使用vue将页面公共部分封装成组件。小程序也使用了组件和模块化思想。

2018.6 - 2018.10

雄通蒲公英以及小程序

该项目是一个供应链服务平台，提供在线报关、仓储、代理进出口、运输、退运返修等服务。有PC端和小程序端。前台用户端主要是提交相应订单报表，数据经公司内部crm系统，与海关报关系统对接。

该项目前后端分离，后台使用TP5来支撑，数据库使用MySQL。前端主要使用vue.js来支撑，页面则采用Bootstrap作为支撑，数据交互使用axios来支撑。小程序使用weui来作为页面的支撑。

我负责前台用户，后台管理员和小程序前端的所有工作，包括页面的搭建，和后台的数据交互等。所有异步请求使用async && await函数封装，后台的权限管理根据请求的数据动态生成页面的菜单和按钮，使用vue将页面公共部分封装成组件。小程序也使用了组件和模块化思想。

2017.4 - 2018.2

华棉所官网以及小程序

该项目为公司官网，是卓尔控股旗下一个B2B棉花电商信息平台，提供棉花批号查询、公检码单列表、物流运输、涉棉企业信息等服务。

本项目后端采用SSM框架来编写，前端用easyUI、jQuery等框架编写静态页面，主要通过form表单提交来进行数据交互，同时还使用easyUI的DataGrid来接受后台数据。小程序则基于MINA框架的WXML、WXSS、JS为代码语言进行开发。使用WEUI框架作为视图层的美化支撑，调用后端的RESTful接口并使用JSON数据进行交互

我主要负责使用HTML/HTML5/CSS3还原设计稿，实现页面静态交互效果，使用ECharts3框架来实现页面数据的可视化展示，持续性对前端页面进行细节性优化，协助后台解决样式bug。
